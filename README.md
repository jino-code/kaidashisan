# 買い出しさん

家族の買い物管理を楽にするための買い物リストアプリです。

## 開発背景

日常の中で家族に何かをお願いする場面は多いと思っています。口頭で伝えたり、メッセージを送ったりと形は様々ですが、いずれの場合も「伝える」という行為とは別に「記録する」という工程が発生しがちです。記録しなければ情報は各自の頭の中やトーク履歴に分散したままになり、買い忘れにつながります。かといって都度メモに書き起こすのは面倒です。

買い出しさんは、家族間の買い物という身近な場面に絞り、LINEで伝えると同時にリストに記録される仕組みを作ることで、「伝える」と「記録する」の二度手間を解消することを目指します。

## コンセプト

「伝えるだけで、記録される」

- 家族はLINEグループに商品名を送るだけ。
- Webアプリでリストを確認・管理し、必要なタイミングでLINEへ共有できる。

## 主要な機能

- LINEグループへのメッセージを買い物リストに自動追加
- 買い物リストの一覧表示・チェック
- 買い物リストの期限付きURLのLINE送信
